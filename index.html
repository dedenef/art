Создание интернет сообщества педагов блогеров

Основная идея нашего сайта заключается в реализации мультимедийного пространства для учителей блогеров, на которм будут доступны основные принципы введения интернет блогов и продвижения себя ка лидера-мнений.

    Наши основные цели

  1. Реализация преподования через онлайн-обучение
  2. Популиризация сетевого блогинга
  3. Неприрываная система образования
  4. Интуитивно понятный интерфейс
  5. Наличие раздельных личных кабинетов
  6. Рейтинговая система обучения

Краткая анатация

Данный проект ориентирован на развитие освоение и продвижение образовательного блогинга – актуальная задача для тех, кто учит сегодняшних школьников. Основными особенностями проекта являются -кадровые ресурсы: коллектив педагогических работников СОШ ФГБОУ «МДЦ «Артек» в количестве 106 чел., привлеченные специалисты из сотрудников «МДЦ «Артек»; - материально-технические ресурсы: помещения школы для проведения совещаний, мастер-классов,оборудование для трансляции вебинаров, для видеозаписи мультимедийных материалов; - методические ресурсы: аналитические материалы; методические рекомендации.
